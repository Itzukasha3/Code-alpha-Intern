🤖 Custom Chatbot with Tkinter GUI

📌 Overview

This is a Python-based chatbot with a Tkinter GUI. It takes user inputs, processes them using predefined commands, and provides responses accordingly. This chatbot is ideal for simple automated conversations and can be extended for more complex use cases.

🚀 Features

GUI-based chatbot using Tkinter 🎨

Predefined command-based responses ✅

Easy-to-use interface 🖥️

Lightweight and fast execution ⚡

🛠️ Technologies Used

Python 🐍

Tkinter (for GUI) 🖼️

Basic command-driven logic 💡

📂 Installation

Clone the repository:

git clone https://github.com/your-username/custom-chatbot.git

Navigate to the project directory:

cd custom-chatbot

Run the chatbot script:

python chatbot.py

▶️ Usage

Launch the chatbot GUI by running chatbot.py.

Type commands or questions in the chatbox.

The chatbot will respond based on predefined logic.
